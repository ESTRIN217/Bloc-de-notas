# Security Policy

## Supported Versions
Actualmente damos soporte a las siguientes versiones:


| Version | Supported          |
| ------- | ------------------ |
| 4.1.0   | :white_check_mark: |
| 4.0     | :x:                |
| < 2.0   | :x:                |

## Reporting a Vulnerability
¡Gracias por ayudarnos a mantener el proyecto seguro! 

1. **No abras un Issue** para reportar una vulnerabilidad.
2. Por favor, utiliza la función de **"Report a vulnerability"** en la pestaña **Security** de este repositorio.
3. Si prefieres el correo, escribe a `gamerestrin217@gmail.com`.
4. Recibirás una respuesta inicial en menos de 3 días hábiles.
